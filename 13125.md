 <h2>一、常用等价无穷小（当 \(x \to 0\)）</h2> <ul>
 <li>\(\sin x \sim x\)</li> <li>\(\tan x \sim x\)</li> <li>\(\arcsin x \sim x\)</li> <li>\(\arctan x \sim x\)</li> <li>\(\ln(1+x) \sim x\)</li> <li>\(e^x - 1 \sim x\)</li>
 <li>\(1 - \cos x \sim \dfrac{1}{2}x^2\)</li>
 <li>\((1+x)^\alpha - 1 \sim \alpha x\)</li>
 </ul>
</div>
<div class="formula-section">
 <h2>二、导数基本公式</h2> <ul>
 <li>\((C)' = 0\)</li>
 <li>\((x^n)' = n x^{n-1}\)</li>
 <li>\((e^x)' = e^x\)</li>
 <li>\((a^x)' = a^x \ln a\)</li>
 <li>\((\ln x)' = \dfrac{1}{x}\)</li>
 <li>\((\log_a x)' = \dfrac{1}{x \ln a}\)</li>
 <li>\((\sin x)' = \cos x\)</li>
 <li>\((\cos x)' = -\sin x\)</li>
 <li>\((\tan x)' = \sec^2 x\)</li>
 <li>\((\arcsin x)' = \dfrac{1}{\sqrt{1 - x^2}}\)</li>
 <li>\((\arccos x)' = -\dfrac{1}{\sqrt{1 - x^2}}\)</li>
 <li>\((\arctan x)' = \dfrac{1}{1 + x^2}\)</li>
 </ul>
</div>
<div class="formula-section">
 <h2>三、求导法则</h2> <ul>
 <li>和差：\((u \pm v)' = u' \pm v'\)</li>
 <li>乘积：\((uv)' = u'v + uv'\)</li>
 <li>商：\(\left( \dfrac{u}{v} \right)' = \dfrac{u'v - uv'}{v^2}\)</li>
 <li>链式法则：若 \(y = f(u),\ u = \varphi(x)\)，则 \(\dfrac{dy}{dx} = \dfrac{dy}{du} \cdot \dfrac{du}{dx}\)</li> </ul></div>
<div class="formula-section">
 <h2>四、基本积分公式</h2> <ul>
 <li>\(\displaystyle \int x^n \, dx = \dfrac{x^{n+1}}{n+1} + C \quad (n \ne 
-1)\)</li>
 <li>\(\displaystyle \int \dfrac{1}{x} \, dx = \ln|x| + C\)</li>
 <li>\(\displaystyle \int e^x \, dx = e^x + C\)</li>
 <li>\(\displaystyle \int a^x \, dx = \dfrac{a^x}{\ln a} + C\)</li>
 <li>\(\displaystyle \int \sin x \, dx = -\cos x + C\)</li>
 <li>\(\displaystyle \int \cos x \, dx = \sin x + C\)</li>
 <li>\(\displaystyle \int \dfrac{1}{1 + x^2} \, dx = \arctan x + C\)</li>
 <li>\(\displaystyle \int \dfrac{1}{\sqrt{1 - x^2}} \, dx = \arcsin x + C\)</li>
 </ul>
</div>
<div class="formula-section">
 <h2>五、积分方法</h2> <ul>
 <li>第一换元法（凑微分）：<br>\(\displaystyle \int f(\varphi(x)) \, \varphi'(x) \, dx = \int f(u) \, du\)</li>
 <li>分部积分法：<br>\(\displaystyle \int u \, dv = uv - \int v \, du\)</li>
 <li>牛顿-莱布尼茨公式：<br>\(\displaystyle \int_a^b f(x) \, dx = F(b) - F(a)\)</
li>
 </ul>
</div>
