## 一、函数与极限

### 1.1 基本极限

```
lim(x→0) sinx / x = 1

lim(x→∞) (1 + 1/x)^x = e

lim(x→0) (1 + x)^(1/x) = e
```

### 1.2 等价无穷小（x→0 时）

| 等价关系 |
|---------|
| sinx ~ x |
| tanx ~ x |
| arcsinx ~ x |
| arctanx ~ x |
| 1 - cosx ~ x²/2 |
| eˣ - 1 ~ x |
| ln(1+x) ~ x |
| aˣ - 1 ~ x·lna |
| (1+x)ᵃ - 1 ~ ax |
| x - sinx ~ x³/6 |
| x - ln(1+x) ~ x²/2 |
| tanx - x ~ x³/3 |

### 1.3 极限运算法则

```
lim[f(x) ± g(x)] = limf(x) ± limg(x)

lim[f(x)·g(x)] = limf(x)·limg(x)

lim[f(x)/g(x)] = limf(x)/limg(x)    (limg(x) ≠ 0)
```

### 1.4 夹逼准则

若 g(x) ≤ f(x) ≤ h(x)，且 limg(x) = limh(x) = A，则 limf(x) = A

### 1.5 单调有界准则

单调有界数列必有极限

## 二、导数与微分

### 2.1 导数定义

```
f'(x) = lim(Δx→0) [f(x+Δx) - f(x)] / Δx
```

### 2.2 基本求导公式

| 函数 f(x) | 导数 f'(x) |
|------------|-------------|
| C（常数） | 0 |
| xⁿ | nxⁿ⁻¹ |
| aˣ | aˣ·lna |
| eˣ | eˣ |
| logₐx | 1/(x·lna) |
| lnx | 1/x |
| sinx | cosx |
| cosx | -sinx |
| tanx | sec²x |
| cotx | -csc²x |
| secx | secx·tanx |
| cscx | -cscx·cotx |
| arcsinx | 1/√(1-x²) |
| arccosx | -1/√(1-x²) |
| arctanx | 1/(1+x²) |
| arccotx | -1/(1+x²) |

### 2.3 求导法则

**四则运算：**

```
[u ± v]' = u' ± v'

[uv]' = u'v + uv'

[u/v]' = (u'v - uv') / v²
```

**链式法则（复合函数）：**

```
[f(g(x))]' = f'(g(x))·g'(x)
```

**反函数求导：**

```
[f⁻¹(x)]' = 1 / f'(y)    (y = f⁻¹(x))
```

### 2.4 隐函数求导

F(x,y) = 0 两边对 x 求导，y 视为 x 的函数，解出 y'

### 2.5 对数求导法

先取对数再求导，适用于幂指函数 y = u(x)^v(x) 和多因子乘除

### 2.6 参数方程求导

```
dy/dx = (dy/dt) / (dx/dt) = ψ'(t) / φ'(t)

d²y/dx² = [d/dt(dy/dx)] / (dx/dt)
```

### 2.7 高阶导数常用公式

```
(xⁿ)⁽ⁿ⁾ = n!

(eᵃˣ)⁽ⁿ⁾ = aⁿ·eᵃˣ

(sinx)⁽ⁿ⁾ = sin(x + nπ/2)

(cosx)⁽ⁿ⁾ = cos(x + nπ/2)

(1/(ax+b))⁽ⁿ⁾ = (-1)ⁿ·n!·aⁿ / (ax+b)ⁿ⁺¹
```

**莱布尼茨公式：**

```
(uv)⁽ⁿ⁾ = Σ(k=0→n) C(n,k)·u⁽ⁿ⁻ᵏ⁾·v⁽ᵏ⁾
```

### 2.8 微分

```
dy = f'(x)dx
```

## 三、中值定理与导数应用

### 3.1 罗尔定理

f(x) 在 [a,b] 连续，(a,b) 可导，f(a)=f(b)，则 ∃ξ∈(a,b)，使 f'(ξ) = 0

### 3.2 拉格朗日中值定理

f(x) 在 [a,b] 连续，(a,b) 可导，则 ∃ξ∈(a,b)，使

```
f'(ξ) = [f(b) - f(a)] / (b - a)
```

### 3.3 柯西中值定理

f(x), g(x) 在 [a,b] 连续，(a,b) 可导，g'(x) ≠ 0，则 ∃ξ∈(a,b)，使

```
f'(ξ)/g'(ξ) = [f(b)-f(a)] / [g(b)-g(a)]
```

### 3.4 洛必达法则

```
lim f(x)/g(x) = lim f'(x)/g'(x)    （适用于 0/0 或 ∞/∞ 型）
```

### 3.5 泰勒公式

```
f(x) = f(x₀) + f'(x₀)(x-x₀) + f''(x₀)/2!·(x-x₀)² + ··· + f⁽ⁿ⁾(x₀)/n!·(x-x₀)ⁿ + Rₙ(x)
```

**拉格朗日余项：** Rₙ(x) = f⁽ⁿ⁺¹⁾(ξ)/(n+1)! · (x-x₀)ⁿ⁺¹

**常用麦克劳林展开（x₀ = 0）：**

```
eˣ = 1 + x + x²/2! + x³/3! + ···

sinx = x - x³/3! + x⁵/5! - ···

cosx = 1 - x²/2! + x⁴/4! - ···

ln(1+x) = x - x²/2 + x³/3 - ···    (-1 < x ≤ 1)

1/(1-x) = 1 + x + x² + x³ + ···    (|x| < 1)

(1+x)ᵃ = 1 + ax + a(a-1)/2!·x² + ···
```

### 3.6 函数单调性

f'(x) > 0 ⇒ f(x) 单调递增；f'(x) < 0 ⇒ f(x) 单调递减

### 3.7 函数极值

**必要条件：** f'(x₀) = 0 或 f'(x₀) 不存在

**第一充分条件：** f'(x) 在 x₀ 左右两侧变号

**第二充分条件：** f'(x₀) = 0 且 f''(x₀) < 0 ⇒ 极大值；f''(x₀) > 0 ⇒ 极小值

### 3.8 曲线的凹凸性与拐点

f''(x) > 0 ⇒ 凹（下凸）；f''(x) < 0 ⇒ 凸（上凸）

f''(x) 在 x₀ 左右两侧变号 ⇒ (x₀, f(x₀)) 为拐点

### 3.9 曲率与曲率半径

```
K = |y''| / (1 + y'²)^(3/2)

R = 1/K
```

### 3.10 渐近线

- **水平渐近线：** y = lim(x→∞) f(x)
- **垂直渐近线：** x = x₀，其中 lim(x→x₀) f(x) = ∞
- **斜渐近线：** y = kx + b，其中 k = lim(x→∞) f(x)/x，b = lim(x→∞) [f(x) - kx]

## 四、不定积分

### 4.1 基本积分公式

| 积分 | 结果 |
|-----|------|
| ∫ xⁿ dx | xⁿ⁺¹/(n+1) + C  (n≠-1) |
| ∫ 1/x dx | ln\|x\| + C |
| ∫ aˣ dx | aˣ/lna + C |
| ∫ eˣ dx | eˣ + C |
| ∫ sinx dx | -cosx + C |
| ∫ cosx dx | sinx + C |
| ∫ tanx dx | -ln\|cosx\| + C |
| ∫ cotx dx | ln\|sinx\| + C |
| ∫ secx dx | ln\|secx+tanx\| + C |
| ∫ cscx dx | ln\|cscx-cotx\| + C |
| ∫ sec²x dx | tanx + C |
| ∫ csc²x dx | -cotx + C |
| ∫ secx·tanx dx | secx + C |
| ∫ cscx·cotx dx | -cscx + C |
| ∫ 1/√(1-x²) dx | arcsinx + C |
| ∫ 1/(1+x²) dx | arctanx + C |
| ∫ 1/(a²+x²) dx | (1/a)·arctan(x/a) + C |
| ∫ 1/√(a²-x²) dx | arcsin(x/a) + C |
| ∫ 1/(a²-x²) dx | 1/(2a)·ln\|(a+x)/(a-x)\| + C |
| ∫ 1/(x²-a²) dx | 1/(2a)·ln\|(x-a)/(x+a)\| + C |
| ∫ 1/√(x²+a²) dx | ln(x + √(x²+a²)) + C |
| ∫ 1/√(x²-a²) dx | ln\|x + √(x²-a²)\| + C |

### 4.2 第一换元法（凑微分法）

```
∫ f[φ(x)]·φ'(x)dx = ∫ f[φ(x)]dφ(x) = F[φ(x)] + C
```

### 4.3 第二换元法

```
∫ f(x)dx  ──(令x=φ(t))──>  ∫ f[φ(t)]·φ'(t)dt
```

常用代换：
- 含 √(a²-x²)：令 x = a·sint
- 含 √(a²+x²)：令 x = a·tant
- 含 √(x²-a²)：令 x = a·sect

### 4.4 分部积分法

```
∫ u dv = uv - ∫ v du
```

**选 u 的优先顺序（LIATE法则）：** 对数 → 反三角 → 代数 → 三角 → 指数

### 4.5 有理函数积分

P(x)/Q(x) 先化成部分分式，再逐项积分

## 五、定积分

### 5.1 定积分定义

```
∫[a→b] f(x)dx = lim(λ→0) Σ(i=1→n) f(ξᵢ)·Δxᵢ
```

### 5.2 牛顿-莱布尼茨公式

```
∫[a→b] f(x)dx = F(b) - F(a) = F(x) |[a,b]
```

### 5.3 定积分性质

```
∫[a→b] f(x)dx = -∫[b→a] f(x)dx

∫[a→a] f(x)dx = 0

∫[a→b] [f(x)±g(x)]dx = ∫[a→b] f(x)dx ± ∫[a→b] g(x)dx

∫[a→b] kf(x)dx = k·∫[a→b] f(x)dx

∫[a→b] f(x)dx = ∫[a→c] f(x)dx + ∫[c→b] f(x)dx
```

**估值定理：** 若 m ≤ f(x) ≤ M，则 m(b-a) ≤ ∫[a→b] f(x)dx ≤ M(b-a)

**积分中值定理：** ∫[a→b] f(x)dx = f(ξ)(b-a)，ξ∈[a,b]

### 5.4 变限积分求导

```
d/dx ∫[α(x)→β(x)] f(t)dt = f(β(x))·β'(x) - f(α(x))·α'(x)
```

### 5.5 奇偶函数的积分

- f(x) 为奇函数：∫[-a→a] f(x)dx = 0
- f(x) 为偶函数：∫[-a→a] f(x)dx = 2∫[0→a] f(x)dx

### 5.6 华里士公式（点火公式）

```
∫[0→π/2] sinⁿx dx = ∫[0→π/2] cosⁿx dx

n为正偶数:  (n-1)/n · (n-3)/(n-2) ··· 3/4 · 1/2 · π/2

n为大于1的正奇数:  (n-1)/n · (n-3)/(n-2) ··· 4/5 · 2/3 · 1
```

### 5.7 定积分的应用

**平面图形面积：**

```
A = ∫[a→b] |f(x) - g(x)|dx
```

**旋转体体积：**

```
绕x轴: Vx = π ∫[a→b] [f(x)]² dx

绕y轴: Vy = 2π ∫[a→b] x|f(x)|dx    （柱壳法）
```

**弧长：**

```
L = ∫[a→b] √(1 + [f'(x)]²) dx

参数方程: L = ∫[α→β] √([x'(t)]² + [y'(t)]²) dt
```

**旋转体侧面积：**

```
S = 2π ∫[a→b] f(x)·√(1 + [f'(x)]²) dx
```

### 5.8 反常积分（广义积分）

```
∫[a→+∞] f(x)dx = lim(b→+∞) ∫[a→b] f(x)dx

∫[a→b] f(x)dx = lim(ε→0⁺) ∫[a+ε→b] f(x)dx    (a 为瑕点)
```

**p-积分收敛条件：**

- ∫[1→+∞] 1/xᵖ dx：p > 1 时收敛，p ≤ 1 时发散
- ∫[0→1] 1/xᵖ dx：p < 1 时收敛，p ≥ 1 时发散

## 六、微分方程

### 6.1 一阶微分方程

**可分离变量方程：**

```
dy/dx = f(x)g(y)  =>  ∫dy/g(y) = ∫f(x)dx + C
```

**齐次方程：**

```
dy/dx = φ(y/x)，令 u = y/x，化为可分离变量方程
```

**一阶线性方程：**

```
y' + P(x)y = Q(x)

通解公式:
y = e^(-∫P(x)dx) · [∫Q(x)·e^(∫P(x)dx)dx + C]
```

### 6.2 可降阶的高阶方程

- **y⁽ⁿ⁾ = f(x) 型：** 逐次积分
- **y'' = f(x, y') 型（不显含 y）：** 令 p = y'，p' = f(x, p)
- **y'' = f(y, y') 型（不显含 x）：** 令 p = y'，y'' = p·dp/dy

### 6.3 二阶常系数线性微分方程

**齐次：** y'' + py' + qy = 0

特征方程 r² + pr + q = 0：

| 判别式 | 特征根 | 通解 |
|-------|--------|------|
| p²-4q > 0 | 相异实根 r₁, r₂ | y = C₁·e^(r₁x) + C₂·e^(r₂x) |
| p²-4q = 0 | 重根 r | y = (C₁ + C₂x)·e^(rx) |
| p²-4q < 0 | 共轭复根 α±βi | y = e^(αx)(C₁cosβx + C₂sinβx) |

**非齐次：** y'' + py' + qy = f(x)

通解 = 齐次通解 + 非齐次特解

**特解设法（待定系数法）：**

- f(x) = Pm(x)·e^(λx)：设 y* = xᵏ·Qm(x)·e^(λx)，k 为 λ 是特征根的重数（0/1/2）
- f(x) = e^(αx)[Pl(x)cosβx + Pn(x)sinβx]：设 y* = xᵏ·e^(αx)[Qm(x)cosβx + Rm(x)sinβx]，m = max(l,n)，k 为 α±βi 是特征根的重数（0/1）

## 七、向量代数与空间解析几何

### 7.1 向量运算

**数量积：** a⃗·b⃗ = |a||b|cosθ

**向量积：** |a⃗×b⃗| = |a||b|sinθ，方向由右手定则确定

**混合积：** [a⃗ b⃗ c⃗] = (a⃗×b⃗)·c⃗

### 7.2 平面方程

- 点法式：A(x-x₀) + B(y-y₀) + C(z-z₀) = 0
- 一般式：Ax + By + Cz + D = 0
- 截距式：x/a + y/b + z/c = 1

### 7.3 直线方程

- 点向式：(x-x₀)/m = (y-y₀)/n = (z-z₀)/p
- 参数式：x = x₀+mt, y = y₀+nt, z = z₀+pt
- 一般式：两平面的交线

### 7.4 距离公式

- 点到平面：d = |Ax₀+By₀+Cz₀+D| / √(A²+B²+C²)
- 点到直线：d = |s⃗×M₀M⃗| / |s⃗|

## 八、多元函数微分学

### 8.1 偏导数

```
fx(x,y) = lim(Δx→0) [f(x+Δx,y) - f(x,y)] / Δx
```

### 8.2 全微分

```
dz = (∂z/∂x)dx + (∂z/∂y)dy
```

### 8.3 复合函数求导（链式法则）

```
∂z/∂x = (∂z/∂u)(∂u/∂x) + (∂z/∂v)(∂v/∂x)
```

### 8.4 隐函数求导

```
F(x,y)=0：dy/dx = -Fx/Fy

F(x,y,z)=0：∂z/∂x = -Fx/Fz，∂z/∂y = -Fy/Fz
```

### 8.5 多元函数极值

**必要条件：** fx(x₀,y₀) = 0，fy(x₀,y₀) = 0

**充分条件：** 令 A = fxx，B = fxy，C = fyy

- AC - B² > 0 且 A < 0 ⇒ 极大值
- AC - B² > 0 且 A > 0 ⇒ 极小值
- AC - B² < 0 ⇒ 非极值
- AC - B² = 0 ⇒ 不确定

### 8.6 拉格朗日乘数法

求 f(x,y) 在 φ(x,y)=0 下的条件极值，构造

```
L(x,y,λ) = f(x,y) + λφ(x,y)

解 ∂L/∂x = 0，∂L/∂y = 0，∂L/∂λ = 0
```

## 九、多元函数积分学

### 9.1 二重积分

**直角坐标：**

```
∬D f(x,y)dxdy = ∫[a→b] dx ∫[φ₁(x)→φ₂(x)] f(x,y)dy
```

**极坐标：** x = rcosθ, y = rsinθ

```
∬D f(x,y)dxdy = ∬D f(rcosθ, rsinθ)·r dr dθ
```

### 9.2 三重积分

- **直角坐标：** ∭Ω f(x,y,z)dxdydz
- **柱坐标：** dxdydz = r dr dθ dz
- **球坐标：** dxdydz = r²sinφ dr dφ dθ

### 9.3 曲线积分

- **第一类（对弧长）：** ∫L f(x,y)ds
- **第二类（对坐标）：** ∫L Pdx + Qdy

### 9.4 格林公式

```
∮L Pdx + Qdy = ∬D (∂Q/∂x - ∂P/∂y) dxdy
```

（L 为 D 的正向边界）

### 9.5 曲面积分

- **第一类（对面积）：** ∬Σ f(x,y,z)dS
- **第二类（对坐标）：** ∬Σ Pdydz + Qdzdx + Rdxdy

### 9.6 高斯公式

```
∯Σ Pdydz + Qdzdx + Rdxdy = ∭Ω (∂P/∂x + ∂Q/∂y + ∂R/∂z) dxdydz
```

### 9.7 斯托克斯公式

```
∮L Pdx + Qdy + Rdz = ∬Σ | dydz  dzdx  dxdy |
                              | ∂/∂x  ∂/∂y  ∂/∂z |
                              |  P     Q     R   |
```

## 十、无穷级数

### 10.1 常数项级数

**等比级数：** Σ(n=0→∞) aqⁿ，|q| < 1 时收敛，和为 a/(1-q)

**p-级数：** Σ(n=1→∞) 1/nᵖ，p > 1 时收敛，p ≤ 1 时发散

### 10.2 审敛法

**比较审敛法：** 0 ≤ uₙ ≤ vₙ，Σvₙ 收敛 ⇒ Σuₙ 收敛；Σuₙ 发散 ⇒ Σvₙ 发散

**比值审敛法（达朗贝尔）：** lim(uₙ₊₁/uₙ) = ρ，ρ < 1 收敛，ρ > 1 发散

**根值审敛法（柯西）：** lim(ⁿ√uₙ) = ρ，ρ < 1 收敛，ρ > 1 发散

**莱布尼茨审敛法（交错级数）：** uₙ 递减且 limuₙ = 0 ⇒ Σ(-1)ⁿuₙ 收敛

### 10.3 绝对收敛与条件收敛

- Σ|uₙ| 收敛 ⇒ Σuₙ 绝对收敛
- Σuₙ 收敛但 Σ|uₙ| 发散 ⇒ 条件收敛

### 10.4 幂级数

```
Σ(n=0→∞) aₙxⁿ

收敛半径: R = lim(n→∞) |aₙ/aₙ₊₁|  或  R = 1/lim(n→∞) ⁿ√|aₙ|
```

### 10.5 常用函数展开式

```
eˣ = Σ(n=0→∞) xⁿ/n!                  x∈(-∞,+∞)

sinx = Σ(n=0→∞) (-1)ⁿx^(2n+1)/(2n+1)!    x∈(-∞,+∞)

cosx = Σ(n=0→∞) (-1)ⁿx^(2n)/(2n)!        x∈(-∞,+∞)

ln(1+x) = Σ(n=1→∞) (-1)^(n-1)xⁿ/n       x∈(-1,1]

1/(1-x) = Σ(n=0→∞) xⁿ                    x∈(-1,1)

(1+x)ᵃ = Σ(n=0→∞) a(a-1)···(a-n+1)/n! · xⁿ    x∈(-1,1)
```

### 10.6 傅里叶级数

```
f(x) = a₀/2 + Σ(n=1→∞)[aₙcos(nπx/l) + bₙsin(nπx/l)]

a₀ = (1/l) ∫[-l→l] f(x)dx

aₙ = (1/l) ∫[-l→l] f(x)cos(nπx/l)dx

bₙ = (1/l) ∫[-l→l] f(x)sin(nπx/l)dx
```

## 十一、常用三角公式

### 11.1 基本恒等式

```
sin²x + cos²x = 1

1 + tan²x = sec²x

1 + cot²x = csc²x
```

### 11.2 倍角公式

```
sin2x = 2sinx·cosx

cos2x = cos²x - sin²x = 2cos²x - 1 = 1 - 2sin²x

tan2x = 2tanx / (1 - tan²x)
```

### 11.3 半角公式

```
sin²(x/2) = (1 - cosx)/2

cos²(x/2) = (1 + cosx)/2
```

### 11.4 和差公式

```
sin(x±y) = sinx·cosy ± cosx·siny

cos(x±y) = cosx·cosy ∓ sinx·siny

tan(x±y) = (tanx ± tany) / (1 ∓ tanx·tany)
```

### 11.5 和差化积

```
sinx + siny = 2sin((x+y)/2)·cos((x-y)/2)

sinx - siny = 2cos((x+y)/2)·sin((x-y)/2)

cosx + cosy = 2cos((x+y)/2)·cos((x-y)/2)

cosx - cosy = -2sin((x+y)/2)·sin((x-y)/2)
```

### 11.6 积化和差

```
sinx·cosy = 1/2[sin(x+y) + sin(x-y)]

cosx·cosy = 1/2[cos(x+y) + cos(x-y)]

sinx·siny = -1/2[cos(x+y) - cos(x-y)]
```
