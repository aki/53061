## 二、导数基本公式

| 函数 $f(x)$ | 导数 $f'(x)$ |
|--------------|----------------|
| $C$ | $0$ |
| $x^n$ | $n x^{n-1}$ |
| $e^x$ | $e^x$ |
| $a^x$ | $a^x \ln a$ |
| $\ln x$ | $\frac{1}{x}$ |
| $\log_a x$ | $\frac{1}{x \ln a}$ |
| $\sin x$ | $\cos x$ |
| $\cos x$ | $-\sin x$ |
| $\tan x$ | $\sec^2 x$ |
| $\arcsin x$ | $\frac{1}{\sqrt{1 - x^2}}$ |
| $\arccos x$ | $-\frac{1}{\sqrt{1 - x^2}}$ |
| $\arctan x$ | $\frac{1}{1 + x^2}$ |

## 三、求导法则

- 和差：$(u \pm v)' = u' \pm v'$
- 乘积：$(uv)' = u'v + uv'$
- 商：$\left( \frac{u}{v} \right)' = \frac{u'v - uv'}{v^2}$
- 链式法则：若 $y = f(u),\ u = \varphi(x)$，则 $\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx}$

---

## 四、基本积分公式

| 被积函数 | 不定积分 |
|----------|-----------|
| $x^n\ (n \ne -1)$ | $\frac{x^{n+1}}{n+1} + C$ |
| $\frac{1}{x}$ | $\ln|x| + C$ |
| $e^x$ | $e^x + C$ |
| $a^x$ | $\frac{a^x}{\ln a} + C$ |
| $\sin x$ | $-\cos x + C$ |
| $\cos x$ | $\sin x + C$ |
| $\frac{1}{1 + x^2}$ | $\arctan x + C$ |
| $\frac{1}{\sqrt{1 - x^2}}$ | $\arcsin x + C$ |

---

## 五、积分方法

- **凑微分**：$\int f(\varphi(x)) \varphi'(x)\,dx = \int f(u)\,du$
- **分部积分**：$\int u\,dv = uv - \int v\,du$
- **牛顿-莱布尼茨**：$\int_a^b f(x)\,dx = F(b) - F(a)$

## 六、经济应用

- 边际成本：$MC = C'(q)$
- 边际收益：$MR = R'(q)$
- 利润最大化：$MR = MC$
- 需求价格弹性：$E_d = \frac{dQ}{dp} \cdot \frac{p}{Q}$
