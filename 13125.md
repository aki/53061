## 一、常用等价无穷小（当 $x \to 0$）

| 无穷小量                | 等价形式             |
|------------------------|----------------------|
| $\sin x$              | $x$                 |
| $\tan x$              | $x$                 |
| $\arcsin x$           | $x$                 |
| $\arctan x$           | $x$                 |
| $\ln(1+x)$            | $x$                 |
| $e^x - 1$             | $x$                 |
| $1 - \cos x$          | $\dfrac{1}{2}x^2$   |
| $(1+x)^\alpha - 1$    | $\alpha x$          |

> ✅ 提示：用于求极限，尤其是 $\dfrac{0}{0}$ 型。

## 二、导数基本公式

| 函数 $f(x)$         | 导数 $f'(x)$                     |
|----------------------|----------------------------------|
| $C$（常数）         | $0$                             |
| $x^n$               | $n x^{n-1}$                     |
| $e^x$               | $e^x$                           |
| $a^x$               | $a^x \ln a$                     |
| $\ln x$             | $\dfrac{1}{x}$                  |
| $\log_a x$          | $\dfrac{1}{x \ln a}$            |
| $\sin x$            | $\cos x$                        |
| $\cos x$            | $-\sin x$                       |
| $\tan x$            | $\sec^2 x$                      |
| $\arcsin x$         | $\dfrac{1}{\sqrt{1-x^2}}$       |
| $\arccos x$         | $-\dfrac{1}{\sqrt{1-x^2}}$      |
| $\arctan x$         | $\dfrac{1}{1+x^2}$              |

## 三、求导法则

| 法则         | 公式                                                                 |
|--------------|----------------------------------------------------------------------|
| 和差         | $(u \pm v)' = u' \pm v'$                                            |
| 乘积         | $(uv)' = u'v + uv'$                                                 |
| 商           | $\left( \dfrac{u}{v} \right)' = \dfrac{u'v - uv'}{v^2}$             |
| 链式法则     | $\dfrac{dy}{dx} = \dfrac{dy}{du} \cdot \dfrac{du}{dx}$              |
| 隐函数求导   | 对方程两边关于 $x$ 求导，$y$ 视为 $x$ 的函数，结果含 $y'$         |
| 对数求导法   | 适用于 $y = u(x)^{v(x)}$，先取 $\ln$ 再求导                         |

## 四、基本积分公式

| 被积函数                     | 不定积分                              |
|------------------------------|----------------------------------------|
| $x^n \quad (n \ne -1)$      | $\dfrac{x^{n+1}}{n+1} + C$            |
| $\dfrac{1}{x}$              | $\ln|x| + C$                          |
| $e^x$                       | $e^x + C$                             |
| $a^x$                       | $\dfrac{a^x}{\ln a} + C$              |
| $\sin x$                    | $-\cos x + C$                         |
| $\cos x$                    | $\sin x + C$                          |
| $\dfrac{1}{1+x^2}$          | $\arctan x + C$                       |
| $\dfrac{1}{\sqrt{1-x^2}}$   | $\arcsin x + C$                       |

---

## 五、积分方法

| 方法               | 公式 / 说明                                                                 |
|--------------------|-----------------------------------------------------------------------------|
| **第一换元法**<br>（凑微分） | $\displaystyle \int f(\varphi(x)) \varphi'(x)\,dx = \int f(u)\,du$        |
| **分部积分法**     | $\displaystyle \int u\,dv = uv - \int v\,du$<br>口诀：“反对幂指三”选 $u$ |
| **牛顿-莱布尼茨公式** | $\displaystyle \int_a^b f(x)\,dx = F(b) - F(a)$                           |

## 六、经济应用（经管类重点！）

| 概念               | 公式                                      | 说明                     |
|--------------------|-------------------------------------------|--------------------------|
| 边际成本（MC）     | $MC = C'(q)$                             | 成本对产量的导数         |
| 边际收益（MR）     | $MR = R'(q)$                             | 收益对产量的导数         |
| 边际利润（ML）     | $ML = L'(q) = MR - MC$                   | 利润对产量的导数         |
| 利润最大化条件     | $MR = MC$ 或 $L'(q) = 0$                 | 一阶条件                 |
| 需求价格弹性 $E_d$ | $E_d = \dfrac{dQ}{dp} \cdot \dfrac{p}{Q}$ | 通常取绝对值分析敏感度   |

> 💡 **考试提示**：经济应用题常以选择题或简单计算题出现，务必掌握“边际=导数”这一核心思想！
